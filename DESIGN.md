# Design

Mood: "morning desk light — unhurried, warm-lit discipline." A calm focus tool. The blocked page is a quiet editorial pause, not a warning; the popup and options are plain, dense, familiar product surfaces.

## Theme

Light only. Pure white ground; warmth lives in the brand colors and the one serif moment, never in the surface.

## Color (OKLCH)

| Token | Value | Hex | Use |
|---|---|---|---|
| `--bg` | `oklch(1 0 0)` | `#ffffff` | Page ground |
| `--surface` | `oklch(0.967 0.005 57)` | `#f7f3f1` | Hover fills, quiet panels |
| `--line` | `oklch(0.90 0.008 57)` | `#e2ddd9` | Hairline borders, dividers |
| `--ink` | `oklch(0.28 0.012 57)` | `#2e2723` | Body text (14.7:1 on bg) |
| `--muted` | `oklch(0.47 0.012 57)` | `#605954` | Secondary text (6.9:1 on bg) |
| `--primary` | `oklch(0.53 0.125 55)` | `#a1540f` | Toasted amber. Buttons, brand mark, focus rings. White text on it (5.5:1) |
| `--primary-deep` | `oklch(0.47 0.115 55)` | `#8a4603` | Button hover |
| `--amber-tint` / `--amber-text` | `oklch(0.93 0.05 65)` / `oklch(0.45 0.12 55)` | `#ffe2c6` / `#863f00` | "Focus hours" status pill (6.2:1) |
| `--pine` / `--pine-tint` | `oklch(0.30 0.07 170)` / `oklch(0.93 0.04 165)` | `#003928` / `#d0f1e1` | "Off hours" pill, success messages (10.7:1) |
| `--error` | `oklch(0.50 0.16 27)` | `#ac312c` | Error text (6.5:1 on bg) |

Strategy: Restrained. Amber is the only action color; pine marks rest/success; everything else is warm-gray neutral.

## Typography

- **UI sans**: `-apple-system, BlinkMacSystemFont, "Segoe UI", system-ui, sans-serif` — everything except the blocked-page headline.
- **Display serif** (blocked page headline only): `ui-serif, "New York", Georgia, "Times New Roman", serif`, 38px/600, letter-spacing -0.015em, `text-wrap: balance`. This is the single deliberate brand moment.
- Product scale (fixed rem-equivalents): 12 / 13 / 14 / 16 / 22. Hierarchy in dense surfaces is carried by weight (650/600/500/400) and ink-vs-muted color, not size jumps.
- No webfonts — extension pages must work offline; system stacks only.

## Components

- **Buttons**: amber fill, white 600 text, radius 7–8px, hover → `--primary-deep`, `focus-visible` outline 2px.
- **Inputs**: white bg, 1px `--line` border, radius 7–8px; focus = amber border + 3px amber ring at 15% alpha. Placeholder = `--muted`.
- **Status pills**: 999px radius, tint bg + deep text (amber = focus hours, pine = off hours / bypass active).
- **Rows** (popup status, options site list): hairline-separated label/value or label/control rows. No card boxes; sections separate by whitespace + 1px `--line` rules.
- **Brand mark**: inline SVG shield `M12 2l8 3v6.09c0 5.05-3.41 9.76-8 10.91-4.59-1.15-8-5.86-8-10.91V5z`, `currentColor` amber.
- **Messages**: `.msg-success` = pine, `.msg-error` = `--error`, reserved-height line below the action.

## Motion

150ms ease transitions on interactive state only (border, background, color, box-shadow). Wrong-password shake (0.35s) on the blocked page and popup; all motion nulled under `prefers-reduced-motion: reduce`.

## Voice

Temporal, never punitive: "twitter.com can wait." / "Focus hours" / "Off hours" / "Need it now? Unlock everything for 15 minutes." No ALL-CAPS, no warning icons, no red except genuine errors.

## Surfaces

- **Popup** (292px): mark + title + status pill header; Schedule / Unblocks-in rows; quiet bypass; Settings footer link.
- **Options** (560px column): mark + title masthead; three hairline sections (Focus hours, Bypass password, Blocked sites).
- **Blocked page**: centered 480px column — shield, serif headline "{site} can wait.", schedule + countdown, hairline, bypass, footer.
